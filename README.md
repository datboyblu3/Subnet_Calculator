# Subnet_Calculator


Inputs are an IP address and a subnet mask and returns the below results:

- Network Address
- Broadcast Address
- Total Number of Hosts
- Number of Usable Hosts
- Usable IP Address range
- Subnet Mask
- IP Class
- Binary Subnet Mask
